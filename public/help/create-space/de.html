Raum erstellen

Ein Raum beschreibt den konkreten Ort innerhalb einer Spielstätte, an dem eine Veranstaltung stattfindet – zum Beispiel ein Saal, ein Raum, eine Bühne oder ein Außenbereich. Lege den Raum möglichst eindeutig an, damit Besucher:innen sofort verstehen, wo das Event stattfindet.

Gib dem Raum einen klaren Namen und ergänze bei Bedarf zusätzliche Informationen wie Lage im Gebäude, Etage oder besondere Hinweise zur Erreichbarkeit. Je genauer die Angaben sind, desto einfacher finden Gäste den richtigen Ort.

Tipps

Verwende kurze, eindeutige Bezeichnungen wie „Großer Saal“, „Studio 2“ oder „Innenhof“. Vermeide unnötig lange oder doppelte Namen, besonders wenn die Spielstätte bereits bekannt ist.

Falls es mehrere Räume gibt, achte auf eine konsistente Benennung, damit sie leicht unterschieden werden können. Zusätzliche Hinweise wie „1. OG“ oder „Eingang rechts“ können sehr hilfreich sein.
